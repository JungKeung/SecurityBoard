# 모두의 게시판

## 🖥️ 프로젝트 설명
"모두의 게시판" 은 일반 유저들이 자유롭게 글을 작성하고 공유할 수 있는 웹 서비스입니다. <br/>
추후 게시판 기능추가와 관리자 기능을 업데이트하여 더 안정적인 서비스를 제공할 예정입니다.

## ✍️ 서비스 기능
1) 회원가입 기능 구현
2) 로그인 및 로그아웃 기능 구현
4) 게시글 등록, 수정, 삭제 기능 구현 
5) 게시물 조회 기능 구현
6) 게시물 검색 기능 구현
